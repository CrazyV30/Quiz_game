# 🇺🇦 Quiz App (C#)

Ukrainian-language console quiz app. Admin creates quizzes, users register and play. Data stored in JSON.

## Features
- 🔐 Built-in admin account
- 👤 Separate user registration
- 📝 Custom quizzes on any topic
- 💾 JSON file storage
- 🖥️ Ukrainian UI

## Run
```bash
git clone https://github.com/CrazyV30/Quiz_game.git
cd quiz-app
dotnet run
